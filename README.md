# Abstract Help Center Clone

A responsive front-end implementation of the Abstract Help Center landing page built with HTML5 and CSS3.

---

## 📌 Overview

This project is a recreation of the Abstract Help Center homepage, focusing on modern layout techniques, clean UI design, and responsive structure. It demonstrates the use of Flexbox, CSS Grid, custom properties (CSS variables), and smooth hover interactions.

---

## ✨ Features

- Modern and clean user interface
- Fully responsive layout
- Header with navigation buttons
- Hero section with search bar
- Feature cards using CSS Grid
- Smooth hover interactions
- Multi-column footer layout
- Reusable CSS variables for easier customization
- Lightweight and dependency-free

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS Variables (Custom Properties)
